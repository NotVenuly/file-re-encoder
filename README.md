# A command line tool made in C++  for File re-encoding


## instructions

* Copy a path to the text file you want to re-encode with or without quotes.

* Run program.exe executable preferably in the console

* Paste in your file path

* Copy a path to the folder you want the JSON file at

* Paste the path into the console

* Create a name for your re-encoded JSON file

* Check your folder to make sure everything worked properly

A correct output is something like {"line_count":2, "lines":["Hello World!", "how are you today?"]}

## Development information

I made this project because I wanted expand my portfolio more into engineering rather than game development.
This project uses no external libraries, and uses fstream include to handle opening, reading and creating files.
Line reading saving and transforming is all written by me.
I also focused on making the code readable and easily understandable to practice proffesional environment programming.
From start until finish the development time was a little under 3 days.
